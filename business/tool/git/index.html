# Git

# 删除已提交到远程的文件

先在 .gitignore 文件里添加需要忽略，然后执行

git rm -r --cached <fileName>
1

将<fileName>整个替换为要删除的文件夹。

该操作不会影响本地的文件

# git rebase

场景：当 dev分支 开发过程中，master也发生了其他commit。需要把这些同步到到 dev 上

操作：

  • 切换到dev分支
  • 右键master - “将当前变更变基到 master”
  • 逐步变基，解决冲突
  • dev分支上即可获得最新的master
